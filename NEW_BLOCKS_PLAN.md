# خطة العناصر الجديدة للصفحة الرئيسية

## 20 عنصر جديد متنوع

### 1. **brands-slider** - عرض العلامات التجارية
- عرض شعارات العلامات التجارية في سلايدر
- قابل للتخصيص (عدد العناصر، السرعة، التوقف التلقائي)

### 2. **testimonials** - آراء العملاء
- عرض تقييمات وآراء العملاء
- دعم الصور والنصوص والتقييمات بالنجوم

### 3. **countdown-banner** - بانر العد التنازلي
- بانر مع عداد تنازلي للعروض
- قابل للتخصيص (التاريخ، النص، الألوان)

### 4. **video-section** - قسم الفيديو
- عرض فيديو تعريفي أو ترويجي
- دعم YouTube و Vimeo

### 5. **instagram-feed** - خلاصة Instagram
- عرض آخر منشورات Instagram
- تكامل مع Instagram API

### 6. **newsletter-popup** - نافذة الاشتراك
- نافذة منبثقة للاشتراك في النشرة البريدية
- قابلة للتخصيص (التوقيت، التكرار)

### 7. **products-tabs** - منتجات بتبويبات
- عرض منتجات مع تبويبات (جديد، مميز، عروض)
- تصميم احترافي مع تأثيرات

### 8. **image-text-split** - صورة ونص منقسم
- قسم بصورة ونص جنباً إلى جنب
- قابل للعكس (صورة يمين/يسار)

### 9. **stats-counter** - عداد الإحصائيات
- عرض إحصائيات المتجر (عملاء، منتجات، طلبات)
- أرقام متحركة مع أيقونات

### 10. **blog-posts** - آخر المقالات
- عرض آخر مقالات المدونة
- تصميم بطاقات احترافي

### 11. **parallax-banner** - بانر Parallax
- بانر مع تأثير Parallax
- قابل للتخصيص بالكامل

### 12. **products-grid-advanced** - شبكة منتجات متقدمة
- عرض منتجات بتصميم Grid متقدم
- فلاتر وترتيب

### 13. **collection-list** - قائمة المجموعات
- عرض مجموعات المنتجات
- تصميم بطاقات مع صور

### 14. **app-download** - تحميل التطبيق
- قسم لتحميل تطبيق الجوال
- روابط App Store و Google Play

### 15. **faq-accordion** - الأسئلة الشائعة
- قسم الأسئلة الشائعة بتصميم Accordion
- قابل للتوسع والطي

### 16. **team-members** - فريق العمل
- عرض أعضاء فريق العمل
- صور ومعلومات التواصل

### 17. **pricing-table** - جدول الأسعار
- عرض خطط الأسعار أو الباقات
- تصميم احترافي مع مقارنة

### 18. **timeline** - الخط الزمني
- عرض تاريخ المتجر أو الإنجازات
- تصميم Timeline احترافي

### 19. **map-section** - قسم الخريطة
- عرض موقع المتجر على الخريطة
- تكامل مع Google Maps

### 20. **social-proof** - إثبات اجتماعي
- عرض إشعارات المبيعات الأخيرة
- تأثير حي للمبيعات
